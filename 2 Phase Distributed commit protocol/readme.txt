Required installation:
-Install Python 3.11


1. To run the project,open terminal and run command - python3 main.py
    and then run command - python3 coordinator.py in separate terminal.
2. This project has coordinator.py,main.py,node.py and vars.py
3. All the required output are shown once you run these commands.
4. If you receive port error then change the port number in vars.py file.
