Required Installation
- Install python3
- pip install rpyc


1. To run the project, run command - python3 server.py in a terminal 
   and then run - python3 client.py in a different terminal
2. This project has 3 folders.
3. For all 3 parts, run the client and server on 2 separate terminals, 
   (if you see port use error on terminal, then change the port number in respective python files.)
4. For all three parts run command python3 server.py in one terminal and in other terminal run
   command python3 client.py
5. Do the operations as per instructions on command prompt.
